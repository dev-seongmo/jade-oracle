외국인도 쉽게 따라오는 사주팔자 튜토리얼. 태어난 국가 시간을 한국시간(KST)으로 변환하고, 오행(木火土金水)을 애니메이션으로 재미있게 배웁니다.

한국시간 변환 시작하기

출생 시간 → 한국시간(KST) 변환기

사주는 기준시를 맞추는 것이 중요합니다. 태어난 국가와 현지 출생시각을 입력하면 한국시간(UTC+9)으로 자동 계산합니다.

  1. Step 1. 현지 출생정보 입력

    국가와 출생 시간을 정확히 입력하면 시차를 반영할 준비가 끝납니다.

  2. Step 2. 한국시간으로 변환

    선택한 국가의 UTC 오프셋을 기준으로 한국시간(UTC+9)으로 자동 보정합니다.

  3. Step 3. GPT Pro AI API 해석

    보정된 시간 기반 사주 원국을 전달해 성향/관계/커리어/운세 흐름을 스토리텔링으로 생성합니다.

  4. Step 4. 오행 밸런스 리포트

    목화토금수 강약을 시각화하고 실천 가이드를 언어별로 제공합니다.
